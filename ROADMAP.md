# OpenBeak Roadmap 🐙🚀

This document outlines the planned trajectory for OpenBeak: Macroctopus Agentaculum.

## v1.0.0: The Hatchling (Initial Release)
- [ ] **Core Engine**: High-concurrency scanner coordination logic.
- [ ] **TUI Foundation**: Stealthy, hacker-themed terminal interface using Bubble Tea.
- [ ] **Hunter Tentacles**: Initial modules for fast network/API reconnaissance.
- [ ] **Reporter Tentacles**: Initial JSON/TUI structured findings output.
- [ ] **Headless CLI**: Standard Cobra subcommands for automation.
- [ ] **Contribution Workflow**: Fully functional `scaffold-tentacle` system for community input.

## v1.1.0: Precise Strike
- [ ] **Advanced Hunter Tentacles**: Modules for probing deeper into misconfigured OpenClaw instances.
- [ ] **SIEM/SOAR Reporters**: Integrate findings with external security platforms.
- [ ] **Dry-Run Safety**: Scoped boundaries to prevent accidental disruptions.

## v2.0.0: The Kraken
- [ ] **Beak Tentacles**: Automated remediation (patching configs, revoking permissions).
- [ ] **MCP Server Support**: Expose OpenBeak as an MCP server for AI-controlled scanning.
- [ ] **Plugin System**: Dynamic loading of external tentacle packages.
